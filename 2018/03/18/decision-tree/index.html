4. CART

CART假设决策树是二叉树，内部结点特征取值为“是”和“否”，递归二分每个特征。

  • 回归数使用平方误差最小化准则，越小越好。
  • 分类数使用基尼指数最小化准则，越小越好。

（4）对$g(t)=\alpha$的内部结点$t$进行剪枝，并对叶结点$t$以多数表决法决定其类，得到树$T$。

（7）采用交叉验证法在子树序列$T_0,T_1,…,T_n$中选择最优子树$T_\alpha$。
